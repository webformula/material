Contains all the style css variables
The theme.css file should be loaded before scripts. You can modify all the colors, shapes, an other variables for both light and dark themes

Color schemes (light, dark)
Change between dark and light color schemes
There are 5 mechanisms that affect the color scheme
  • Local storage: mdw-color-scheme = 'light | dark' (user preference setting)
  • CSS class: .mdw-theme-dark
  • CSS :root { color-scheme: light | dark; } (this is not officially supported ina ll browsers)
  • CSS media: prefers-color-scheme (browser / system setting)
  • mdwUtil: mdwUtil.toggleColorScheme('' | 'light | dark')
      
  ${`// Local storage. User preference storage
    
  window.localStorage.setItem('mdw-color-scheme', 'light | dark');`}
