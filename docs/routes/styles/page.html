Color schemes (light, dark)
Change between dark and light color schemes
There are 5 mechanisms that affect the color scheme
  • Local storage: wfc-color-scheme = 'light | dark' (user preference setting)
  • CSS class: .wfc-theme-dark
  • CSS :root { color-scheme: light | dark; } (this is not officially supported ina ll browsers)
  • CSS media: prefers-color-scheme (browser / system setting)
  • wfcUtil: wfcUtil.toggleColorScheme('' | 'light | dark')
Using theme css color variable will auto adjust with light and dark themes
dark_mode light_mode

  window.localStorage.setItem('wfc-color-scheme', 'light | dark');`}

  wfcUtil.toggleColorScheme('dark');`}

Using theme variables
You can use the theme.css variables in your css classes to modify colors
Tones 0 4 6 10 12 17 20 22 24 30 40 50 60 70 80 87 90 92 94 95 96 98 100
Alphas 0 4 5 6 8 10 11 12 16 20 26 38 60 76
        
  ${`/* use theme color */
  .someclass {
    color: var(--wfc-primary);
    /* use custom theme color */
    color: var(--wfc-custom-color);
  }
  
  /* use theme color tone */
  .someclass {
    color: var(--wfc-primary-10);
    /* use custom theme color */
    color: var(--wfc-custom-color-10);
  }
  
  /* use theme color with alpha */
  .someclass {
    color: var(--wfc-primary-alpha-76);
  }`}
