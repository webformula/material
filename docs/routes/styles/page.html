Theme file (theme.css)
Contains all the style css variables
The theme.css file should be loaded before scripts. You can modify all the colors, shapes, an other variables for both light and dark themes

Color schemes (light, dark)
Change between dark and light color schemes
There are 5 mechanisms that affect the color scheme
  • Local storage: mdw-color-scheme = 'light | dark' (user preference setting)
  • CSS class: .mdw-theme-dark
  • CSS :root { color-scheme: light | dark; } (this is not officially supported ina ll browsers)
  • CSS media: prefers-color-scheme (browser / system setting)
  • mdwUtil: mdwUtil.toggleColorScheme('' | 'light | dark')
Using theme css color variable will auto adjust with light and dark themes
dark_mode light_mode

Using theme variables
You can use the theme.css variables in your css classes to modify colors
Tones 0 4 6 10 12 17 20 22 24 30 40 50 60 70 80 87 90 92 94 95 96 98 100
Alphas 0 4 5 6 8 10 11 12 16 20 26 38 60 76
        
  ${`/* use theme color */
  .someclass {
    color: var(--mdw-primary);
    /* use custom theme color */
    color: var(--mdw-custom-color);
  }
  
  /* use theme color tone */
  .someclass {
    color: var(--mdw-primary-10);
    /* use custom theme color */
    color: var(--mdw-custom-color-10);
  }
  
  /* use theme color with alpha */
  .someclass {
    color: var(--mdw-primary-alpha-76);
  }`}
Generating theme at runtime
        
  ${`// add to build script
    
  import generate from '@webformula/material/themeGenerator';

  // generate(input path, output path)
  generate('./src/theme.css', './docs/theme.css');
  
  // use default theme.css file from @webformula/material
  generate('./node_modules/@webformula/material/src/theme.css', './docs/theme.css');`}
